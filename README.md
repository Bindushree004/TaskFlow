# ✅ TaskFlow – Daily Task Tracker

TaskFlow is a modern web-based Daily Task Tracker that helps users organize their daily activities efficiently. It allows users to add, search, filter, and manage tasks with priorities, due dates, categories, and completion tracking.

---

## 🚀 Features

- User Login
- Add New Tasks
- Search Tasks
- Filter Tasks (All, Pending, Completed)
- Set Due Date & Time
- Task Priority (High, Medium, Low)
- Task Categories
- Mark Tasks as Completed
- Delete Tasks
- Clear Completed Tasks
- Delete All Tasks
- Task Statistics
- Progress Bar
- Reminder Notification Sound
- Responsive User Interface

---

## 🛠️ Tech Stack
